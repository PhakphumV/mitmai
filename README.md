# mitmai
A LINE Chatbot to check if an URL is legit or phising.
น้องแมวแชทบอทบน LINE คอยเช็คลิงค์ที่พี่ๆส่งมาว่าเป็นลิงค์มิตรหรือมิจกันแน่
